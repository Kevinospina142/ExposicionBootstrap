Historia

Fue diseñado por Mark Otto y Jacob Thornton, ambos en Twitter crearon el Framework Bootstrap para usarlo internamente en los proyectos de la compañía

¿Qué es?

Bootstrap es un framework de CSS, este es utilizado en el área de desarrollo FrontEnd, va en conjunto de JS y CSS para darle una apariencia e interactividad más creativa al estructurado HTML y crear sitios responsive.

Elementos

  • Barras de Navegación
  • Menús Despegables
  • Migas de Pan
  • Paginación
  • botones
  • Glyphicons

Ventajas y Desventajas

Ventajas

  • Elementos adaptativos
  • Buen respaldo y comunidad para su documentación
  • Permite crear sitios responsive
  • Los estilos son faciles de utilizar

Desventajas

  • Suele tener mucho código para algunos de sus elementos
  • Si no se modifica, los sitios web que utilicen bootstrap seran muy similares
  • Requiere código HTML de mas para poder utilizar sus elementos

Elementos nuevos en Bootstrap 5

  • Se quito la dependencia forzada de JQUERY
  • Se elimino la compatibilidad con internet Explorer en versiones 10 y 11
  • Se elimino la compatibilidad con:
    • Microsoft Edge Legacy
    • Firefox V.60 e inferiores
    • Safari y IOS V.10 e inferiores
    • Android V.6 e inferiores(versión Marshmallow
  • Nuevo acordeon
  • Etiquetas Flotantes
  • Marcador de Posición
  • Soporte RTL
  • Formularios Actualizados
  • Paleta de Colores
  • Libreria de Inconos SVG
